# Process Guard 1.0.2

## Sentinel Improvements

- Removed the redundant custom close button from the Sentinel chat toolbar
- Replaced text plus/minus buttons with compact line-icon size controls
- Kept the Fit control for quickly restoring a usable chat size
- Added Included, Free Tier, Standard, and Premium labels to suggested models
- Reordered genuine free/included choices first where providers offer them
- Reordered lower-cost/default choices first where no permanent free API model is guaranteed
- Added clear billing, quota, account-access, and premium-model guidance
- Added an asynchronous engine/model connectivity test before Save + Restart
- Prevented staged API keys and provider settings from being committed when validation fails

## Documentation And Distribution

- Added application screenshots to the GitHub README
- Updated setup and getting-started guidance for model tiers and validation
- Updated application, installer, portable executable, and source package to version 1.0.2
